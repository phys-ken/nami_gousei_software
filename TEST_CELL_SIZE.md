# ブラウザテスト指示書 — 1目盛サイズ機能 + 退行確認

**対象エージェント**: ブラウザを操作できる LLM Agent
**テスト URL**: http://localhost:8000
**前提**: `python server.py` が起動中であること
**目的**:
1. 新機能「1目盛サイズ（cellSize）の任意指定」が正しく動作することを確認する
2. cellSize 未設定時、**従来の出力（580×200px Canvas）と完全に同一**であることを確認する
3. 既存機能（Type1〜5・PDF・ZIP・スタイルプリセット・編集操作）に退行がないことを確認する

---

## 重要な注意

- **既存出力の保全**: 設定欄を一度も触らない場合、画像出力ピクセル寸法は**従来通り 580×200**でなければなりません。1pxでも違ったら不合格です。
- **localStorage**: テスト前に必要に応じてブラウザの localStorage（キー: `waveapp_cellSize`）をクリアしてください。
- 各テストでブラウザの開発者コンソール（F12）を開き、JS エラーが発生していないかも確認すること。

---

## 事前準備

1. ブラウザで http://localhost:8000 を開く
2. ページが正常に表示されること（白画面・JS エラーがないこと）を確認
3. 左パネル「グリッド設定」セクションを見つける
4. 「1目盛サイズ（任意）」という折りたたみ（▸）が表示されることを確認
5. 折りたたみは初期状態で**閉じている**ことを確認

---

## 参考: 主要な計算式

```
canvas.width  = (xMax - xMin) * cellPxW + paddingLeft  + paddingRight   (cellPxW 指定時)
              = 580                                                       (未指定時)
canvas.height = (yMax - yMin) * cellPxH + paddingTop   + paddingBottom  (cellPxH 指定時)
              = 200                                                       (未指定時)

paddingLeft = paddingRight = 52
paddingTop  = 32, paddingBottom = 44

問題出力Canvas（pixelRatio=2）:
  canvas.width(physical)  = canvas.width(logical) × 2
  canvas.style.width      = canvas.width(logical) px

エディタ Canvas（pixelRatio=1）:
  canvas.width(physical)  = canvas.width(logical)
  canvas.style.width      = canvas.width(logical) px
```
