## 0. 退行確認（既存 Type1〜5 が壊れていないこと）

### 0-1. Type 1 基本動作

1. 設定パネル → 波 A で速さ `1`、方向 `→` を確認
2. 「波形編集」タブ → 波 A に適当な波形を描く（例: x=2 に y=2、x=4 に y=0 など）
3. 「設問作成」タブ → タイプ `Type 1`、解答時刻 `t=2` → 「設問を生成」
4. ✅ 問題・解答の Canvas が表示される。解答欄に t=0〜2 の進行波が含まれること

### 0-2. Type 3 選択肢モード

1. タイプを `Type 3`、地点 `x=3`、tMax=`6` に設定
2. 「選択肢モード」をON → 選択肢数 `4` → 不正解② ③ に適当な波形を入力
3. 「設問を生成」 → 「問題 PDF」「解答 PDF」「画像 ZIP」を順にダウンロード
4. ✅ 問題 PDF に選択肢が 2 列タイル表示されること
5. ✅ 解答 PDF に「★ 正答」マークが付いた選択肢があること
6. ✅ ZIP に `choice_1.png`〜`choice_4.png` と `_correct` サフィックスのファイルがあること

### 0-3. Type 4 / Type 5

1. 「＋ 波 B を追加」 → 波 B を描く → Type 4 で合成波を生成
2. ✅ 問題・解答 Canvas が正常に表示される
3. Type 5 で範囲 `t=1〜4` を生成 → ✅ 4 枚の解答 Canvas が表示される

## 1. 反射波モードの有効化・無効化

### 1-1. 有効化

1. 左サイドバーの「＋ 反射波モードを追加」ボタンをクリック
2. ✅ ボタンが非表示になり、「反射波設定」セクションが表示される
3. ✅ 設問タイプの `Type 6`・`Type 7` が選択可能（enabled）になっていること
4. ✅ `Type 1`・`Type 2`・`Type 3` が選択不可（disabled）になっていること
5. ✅ 設問タイプのドロップダウンが自動的に `Type 6` に切り替わっていること

### 1-2. 無効化

1. 「反射波設定」セクションの「削除」ボタンをクリック
2. ✅ 「＋ 反射波モードを追加」ボタンが復活する
3. ✅ 「反射波設定」セクションが非表示になる
4. ✅ `Type 1`〜`Type 3` が有効、`Type 6`・`Type 7` が無効に戻ること

### 1-3. localStorage 永続化

1. 反射波モードを有効化 → 境界 `x=7` に変更 → ブラウザをリロード
2. ✅ 反射波モードが有効状態で復元されること
3. ✅ 境界が `x=7` のままであること

## 4. プレビュー（進行波プレビュータブ）

### 4-1. 反射波モード時のプレビュー

1. 反射波モードを有効化、波 A に波形を描く（例: 山が x=2〜4）
2. 「進行波プレビュー」タブ → 「更新」
3. ✅ `t=0`〜`t=5` の各コマに Canvas が表示されること

   - 境界線（破線）

   - 反射波（破線、媒質内のみ）

   が表示されること

### 4-2. 自由端 vs 固定端の視覚的違い

## 5. Type 6 — 指定時刻の合成波

### 5-1. 基本生成

1. 反射波モードを有効化（自由端、境界 x=5）
2. 波 A を描く（例: 右向き、速さ 1、x=1 に y=2、x=3 に y=0）
3. 「設問作成」タブ → `Type 6`、解答時刻 `t=3` → 「設問を生成」
4. ✅ 【問題】に 2 枚の Canvas が表示される：
   - 1 枚目: `t=0` の入射波参照（反射波・合成波なし、境界線あり、灰色領域あり）
   - 2 枚目: 解答欄（`t=3 [s]（解答欄）`ラベル、波なし、境界線あり、灰色領域あり）
5. ✅ 【解答】に 1 枚の Canvas が表示される（入射波＋反射波＋合成波すべて、凡例あり）
6. ✅ 【解説】セクション「各時刻の入射波・反射波・合成波の様子」が表示される
7. ✅ 解説に `t=0`〜`t=3` の計 4 枚の Canvas が表示される

### 5-2. 境界線・灰色領域の確認

1. 生成された全 Canvas（問題・解答・解説）を目視確認
2. ✅ すべての Canvas に灰色領域と境界線が表示されること
3. ✅ 境界線に「媒質の端」ラベルが付いていること
4. ✅ 灰色は印刷時にグレーに見える濃さ（#D0D0D0）であること

### 5-3. 固定端での生成

1. 「固定端」に切り替えて再度生成
2. ✅ 解答 Canvas で合成波が `x=boundary` の地点で y=0 になっていること

### 5-4. PDF 出力

1. 「問題 PDF」をダウンロード → ✅ 問題文・Canvas が含まれる（選択肢なし）
2. 「解答 PDF」をダウンロード → ✅ 解答 Canvas が含まれる
3. ✅ PDF 内の全 Canvas に灰色領域・境界線が表示されること

### 5-5. ZIP 出力

1. 「画像 ZIP」をダウンロード → 展開
2. ✅ `question_1.png`（t=0 参照）、`question_2.png`（解答欄）が含まれる
3. ✅ `answer_1.png`（解答）が含まれる
4. ✅ 各画像に灰色領域・境界線が表示されること
